# Phone-to-PC Flow

This is the practical bridge for your phone.

## What works now

- You can trigger a GitHub Actions workflow from your phone.
- The workflow runs on your self-hosted Windows runner.
- `open-codex` opens Chromium on the desktop.

## Fastest setup path

1. Commit and push this repo to GitHub.
2. Open the repository on your phone.
3. Run the `Phone Trigger Self Hosted` workflow.
4. Choose `open-codex` or `status-check`.

## Suggested next step

- Keep the runner online when you want the phone trigger to control local software.
